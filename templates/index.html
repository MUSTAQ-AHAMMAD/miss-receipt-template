Payment File — Expected Columns
The payment file must contain at least these three columns (exact names are flexible): Sales Order Number   Payment Method   Amount
The Sales Order Number must match the Sales Order Number column in the AR Invoice CSV. Payment Method values are normalised (e.g. MADA, Mada, mada all map to Mada). Each matched method is then mapped to BANK_CHARGES.csv to calculate the correct Misc Receipt amount.
Sales Lines
Odoo POS order lines export — product name, barcode, quantity, subtotal, order ref (CSV or XLSX)
Required

Payment Lines
Odoo POS payments export — order ref, payment method, amount (CSV or XLSX)
Required
 CSV / XLSX
Expected Column Names
Sales Lines: Order Lines/Order Ref Order Lines/Product/Name Order Lines/Product/Barcode Order Lines/Base Quantity Order Lines/Subtotal w/o Tax
Payment Lines: Order Ref Payments/Payment Method Payments/Amount
Column names are matched flexibly — alternative names are also accepted. The AR Invoice total is matched against the uploaded Sales Lines sheet total automatically.
